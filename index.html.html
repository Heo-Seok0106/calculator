계산기 사용 설명서

  • C: 가장 마지막에 입력한 내용을 지우는 Cancel 기능입니다. 연산 도중 숫자 혹은 기호를 잘못 입력한 경우에 이 버튼을 누릅니다. 예를 들어, '13 + 6'을 계산할 때 실수로 6이 아닌 5를 입력한 경우, C를 누르면 '13 +'까지만 입력한 상태로 되돌아갑니다.
  • log: 밑이 10인 상용로그 연산을 하는 기능입니다. 연산을 할 숫자를 누르고 log 버튼을 누르면 계산결과가 나옵니다. logx = y라면, y^10 = x의 형식으로 계산됩니다. 예를 들어, '100'을 누르고 log 버튼을 누르면 '2'가 나옵니다. 10^2 = 100이기 때문입니다.
  • 1/x: 입력한 숫자를 역수를 취하여 소수로 나타내주는 기능입니다. 연산을 할 숫자를 누르고 1/x 버튼을 누르면 계산결과가 나옵니다. 예를 들어, '5'를 누르고 1/x 버튼을 누르면 5의 역수인 1/5이 소수로 표현된 '0.2'가 나타납니다.
  • n!: 1부터 입력한 숫자까지의 모든 자연수의 곱의 연산을 하는 팩토리얼 연산 기능입니다. 연산을 할 숫자를 누르고 n! 버튼을 누르면 계산결과가 나옵니다. 예를 들어, '1'을 누르면 1! = 1이므로 '1'이 나오고, '3'을 누르면 3! = 3 * 2 * 1 = 6이므로 '6'이 나옵니다.
  • 주의! AC: 전체 입력한 내용을 지우는 All Cancel 기능입니다. / C: 가장 마지막에 입력한 내용을 지우는 Cancel 기능입니다.
